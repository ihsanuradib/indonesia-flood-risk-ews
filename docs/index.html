Indonesia Flood Risk EWS

WebGIS Early Warning System dan Flood Inundation Mapper untuk risiko banjir di Indonesia — implementasi awal pada DAS Bengawan Solo, terinspirasi oleh USGS Flood Inundation Mapper.

Sumber Data

Tinggi muka air, debit, dan curah hujan real-time dari BBWS Bengawan Solo (hidrologi.bbws-bsolo.net). Status siaga dihitung dari ambang elevasi hijau / kuning / merah tiap pos.

Pemodelan Genangan

Genangan mengacu pada geometri sungai nyata dari OpenStreetMap di sekitar titik AWLR, di-buffer sesuai tinggi muka air sehingga mengikuti alur sungai sebenarnya

Arsitektur

Backend Python FastAPI (data + model genangan) · Frontend Leaflet
